# Customer Segmentation using Machine Learning

## Overview

This project performs customer segmentation using K-Means Clustering.

## Features

- Data Cleaning
- Exploratory Data Analysis
- Feature Scaling
- K-Means Clustering
- Elbow Method
- Silhouette Score
- PCA Visualization
- Interactive Plotly Charts
- Streamlit Dashboard
- Customer Recommendation Engine

## Tech Stack

- Python
- Pandas
- NumPy
- Scikit-Learn
- Plotly
- Matplotlib
- Seaborn
- Streamlit
